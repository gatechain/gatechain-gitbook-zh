# GateChain 开发者指南

欢迎阅读 GateChain 开发者文档。本指南将帮助您了解 GateChain 的架构以及如何在平台上构建应用程序。

## 概述

GateChain 是一个高性能的区块链平台，支持：
- 智能合约开发
- 跨链应用
- DeFi 协议
- NFT 平台
- 企业解决方案

## 智能合约开发

### 支持的标准
- ERC20 
- ERC721 
- ERC1155

### 开发工具
- 支持 Truffle/Hardhat
- 兼容 Web3.js/Ethers.js
- 开发 IDE 插件
- 测试框架

## API 集成

### RPC 节点
- HTTP API
- WebSocket API
