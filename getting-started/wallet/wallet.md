# GateChain 钱包指南

GateChain 为从个人到企业的用户提供全面的钱包解决方案。本指南介绍了可用的钱包选项及其主要功能。

## 可用钱包

### 1. Web 钱包
- 官方网页界面：[wallet.gatechain.io](https://www.gate.io/zh/web3)
- 完整访问所有 GateChain 功能
- 安全的密钥管理
- 支持所有主流浏览器

### 2. 移动钱包
- 原生 Android 和 iOS 应用
- 安全的生物识别认证
- 交易推送通知
- 支持支付二维码
